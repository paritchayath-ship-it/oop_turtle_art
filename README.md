Overview

This project is an Object-Oriented Python program that generates 9 different turtle-graphics art patterns.
When the program runs, it asks the user to choose a number between 1 to 9, and the corresponding artwork will be displayed.

Each art piece uses random values, so every run produces a unique variation of the pattern.
How to Run the Program
1. Install Python

Python 3.8+ recommended.

2. Run the program
python main.py

3. Choose an art

You will see:

Which art do you want to generate? Enter a number between 1 to 9 inclusive:


Enter a number:

1 → Polygon + inner polygon

2 → Rainbow spirals

3 → Random stars

4 → Rotating squares

5 → Floating circles

6 → Starburst lines

7 → Flower pattern

8 → Radiating lines

9 → Hypnotic circles
